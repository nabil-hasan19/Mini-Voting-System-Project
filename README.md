# Mini Voting System in C

A simple console-based **Mini Voting System** developed using the C programming language. This project simulates a basic election process where users can vote for one of three political parties, and the program automatically counts the votes and announces the winner.

## Features

- Cast votes for three political parties
- Real-time vote counting
- Displays the winning party
- Calculates the winning margin
- Simple menu-driven interface
- Beginner-friendly C project

## Technologies Used

- C Programming Language
- Standard C Library (`stdio.h`)

## How It Works

1. Run the program.
2. Select a party by entering its corresponding number.
3. Continue voting until the election result option is selected.
4. The program displays:
   - Total votes for each party
   - Winning party
   - Winning margin
